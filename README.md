## Running locally

Clone the project

```bash
  git clone https://github.com/GuilhermeToso/Portfolio.git
```

Enter in the project directory

```bash
  cd Portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Stack

**Front-end:** Next.js, React, Material

**Back-end:** Express, Mongoose

**Database:** MongoDB
